# LAST SALVAGE — 게임 기획서

## 개요

| 항목 | 내용 |
|------|------|
| **게임명** | LAST SALVAGE |
| **장르** | 모바일 픽셀 로그라이크 전투 |
| **기술 스택** | Phaser.js (HTML5 Canvas) |
| **배포** | 웹앱 / 모바일 브라우저 (PWA) |
| **주인공** | 스크래퍼 (폐품 수집가) |
| **세계관** | 포스트아포칼립스 폐허. 좀비와 변종 생물로 가득 찬 세계에서 혼자 살아남는 이야기 |

---

## 화면 구조

```
┌──────────────────────────┐
│                          │
│       전투 뷰 (58%)      │
│  캐릭터 자동 진행 + 전투  │
│  배경 패럴랙스 스크롤     │
│                          │
├──────────────────────────┤  ← 픽셀 장식띠 경계
│                          │
│      합성 허브 (42%)     │
│  아이템 합성 / 스킬 강화  │
│  능력치 업그레이드        │
│                          │
└──────────────────────────┘
```

- 전투는 항상 자동 진행 — 허브 조작 중에도 멈추지 않음
- HP 위험 시 전투 뷰 테두리 붉게 펄스 → 자연스럽게 시선 유도
- 위쪽: 배경 스크롤 제스처 / 아래쪽: 탭 스와이프 — 제스처 영역 명확히 분리

---

## 핵심 게임 루프

```
전투 → 폐품/파츠 드롭 → 합성 허브에서 무기 강화/제작 → 더 강한 전투
```

---

## 합성 허브 탭 구조

| 탭 | 아이콘 | 핵심 기능 |
|----|--------|-----------|
| 합성 | 🔧 렌치 | 드롭 파츠 조합 → 무기/장비 제작 및 강화 |
| 스킬 | ⚡ 번개 | 보유 스킬 레벨업 · 배치 슬롯 편집 |
| 능력치 | ❤️ 하트 | 코인 소모로 체력/공격력/방어력 업그레이드 |
| 인벤 | 🎒 가방 | 전체 보유 아이템 열람 · 장착 · 버리기 |

- 탭 터치 + 좌우 스와이프 병행
- 재료 충족 / 스킬 포인트 적립 시 탭에 픽셀 뱃지 알림

## 로그라이크 메카닉

| 메카닉 | 설명 |
|--------|------|
| **유언 시스템** | 사망 시 보유 아이템 1개를 "유산"으로 선택 → 다음 런에 들고 시작 |
| **적이 기억한다** | 이전 런에서 많이 당한 속성에 다음 런 해당 적이 내성 보유 → 전략 다변화 강제 |
| **합성 히스토리 보드** | 발견한 합성 레시피가 도감에 영구 기록 → 수집 욕구 + 콘텐츠 볼륨 확장 |

### 배경 패럴랙스 레이어

| 레이어 | 스크롤 속도 | 내용 |
|--------|-------------|------|
| L1 | 0.1× | 흐린 하늘, 연기, 붕괴된 고층빌딩 실루엣 |
| L2 | 0.3× | 폐공장, 고가도로 잔해 실루엣 |
| L3 | 0.6× | 뒤집힌 차량, 가로등 잔해, 잡초 |
| L4 | 1.0× | 지면 — 캐릭터가 달리는 노면 |

### 캐릭터

- 픽셀 크기: **32×32px** (모바일 2× 스케일 → 실제 64×64 렌더링)
- 필수 애니메이션: 걷기(4f) / 공격(3f) / 피격(2f) / 사망(4f)
- 무기 교체 시 공격 프레임만 스왑하는 모듈 구조

### UI 허브 컨셉

- **트럭 짐칸 작업대** 테마
- 금속 볼트 테두리, 닳은 페인트 질감
- 폰트: 타이틀·수치는 픽셀 폰트 / 설명 텍스트는 일반 폰트 혼용
- 전투 애니메이션은 8~12fps (레트로 감성 유지)
